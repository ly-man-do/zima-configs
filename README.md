# ZimaOS-Docker-Compose-Files
Custom Docker Compose Files for ZimaOS

These are custom written ```docker-compose``` files for a few app that aren't in the ZimaOS App Store.
- ```rclone``` - Rclone syncs files and directories to and from different cloud storage providers. (This file is configured to include Rclone UI)
- ```zurg``` - A self-hosted Real-Debrid webdav server written from scratch.

This also includes custom app image icons
- rclone
- zurg
- ssh

and avatars.
